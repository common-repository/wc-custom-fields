== Description ==

Plugin adds custom WooCommerce fields. Add unlimited fields through pro version: <a href ="https://ziscom.today/product/woocommerce-custom-fields/">Buy Pro Version</a>

== Installation ==

1. Upload the entire `wcf-custom-fields` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

How to use?

* Go to Wordpress menu 'Woocommerce Custom Fields' after activating plugin. 
* Enter new field names and Submit. 
* Go to WooCommerce product page or variations and you will see these fields there.
* You can enter value in each field differently for each product and variation. It gets saved in database.
* Product field names are common for all products however for each product their entry saves differently under different product's post id. 
* Add unlimited fields through pro version: <a href ="https://ziscom.today/product/woocommerce-custom-fields/">Buy Pro Version</a>. Pro version has been integrated with Shoppingfeed plugin.
